# Homeless Kittens  
An excercise in C++. A 2-D real time puzzle game, where your task is to house the poor homless kittens with the right owners! Written in C++ using [SFML](https://www.sfml-dev.org/).  
## How to Run  
Make sure you have SFML installed on your computer. If you are on **linux** then simply type 
```bash
sudo apt install libsfml-dev
```
If you are on **windows** follow [this](https://www.sfml-dev.org/download/) 
Finally in the root project directory  
```bash
make
./main
```
And you will start playing.
